# BlackJack ♠️

A console based BlackJack game built in C# as a learning project.

## Features
- Hit and Stand
- Betting system with chips
- Dealer AI
- Blackjack pays 1.5x
- Ace handling (1 or 11)

## How to play
- Run the program
- Enter your name
- Place your bet
- Type h to hit, s to stand

## Built with
- C#
